# `eda.py` — Exploratory Data Analysis

**Path:** `src/eda.py`

Statistical analysis and visualisation helpers for match data exploration.

**Key functions:** `plot_goal_distribution(df)` — histogram of goals per match, Poisson fit overlay. `plot_home_advantage(df)` — home win/draw/away win rates over time. `plot_league_trends(df, league)` — scoring rates over seasons.

Primarily used in Jupyter notebooks for dataset understanding before modeling.
